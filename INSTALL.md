# מדריך התקנה - HereForYou v0.5.0

## 🚀 הורדה והרצה מהירה

### אפשרות 1: הרצה ישירה (מומלץ למפתחים)

1. **פתח את התיקייה:**
   ```
   C:\Disk1\HereForYou\bin\Release\net10.0-windows10.0.19041.0\win-x64\publish
   ```

2. **הפעל את הקובץ:**
   - לחץ כפול על `HereForYou.exe`
   - האפליקציה תתחיל מיד!

3. **מיקום מסד הנתונים:**
   - הנתונים נשמרים ב: `%LOCALAPPDATA%\Packages\...\LocalState\hereforyou.db3`
   - כל ההגדרות, זיהויים, והתראות נשמרים אוטומטית

---

### אפשרות 2: התקנה למשתמש קצה

#### דרישות מקדימות
- Windows 10 version 19041 ומעלה (Windows 10 May 2020 Update)
- .NET 10 Runtime (יותקן אוטומטית עם האפליקציה)

#### שלבי התקנה

1. **העתק את תיקיית publish למיקום קבוע:**
   ```
   העתק את: C:\Disk1\HereForYou\bin\Release\net10.0-windows10.0.19041.0\win-x64\publish
   אל: C:\Program Files\HereForYou
   ```

2. **צור קיצור דרך על שולחן העבודה:**
   - לחץ ימין על `HereForYou.exe`
   - בחר "שלח אל" → "שולחן העבודה (צור קיצור דרך)"
   - שנה שם: "HereForYou - נוכחות הורית"

3. **הרצה ראשונית:**
   - לחץ על הקיצור
   - אפשר גישה אם Windows מבקש הרשאות

## 🎯 שימוש ראשון

### 1. הפעלת ניטור
- לחץ על כפתור **"התחל ניטור"**
- האפליקציה תתחיל להאזין (במצב mock) ולעקוב אחר זמן המסך

### 2. התראות
- כל 10-30 שניות הסימולציה תזהה "אמא" או "אבא"
- אחרי 3 פעמים (ברירת מחדל) תקבל התראה
- התראה קריטית (5+ פעמים) = מסך אדום מלא + רטט

### 3. הגדרות
- עבור לטאב **"הגדרות"**
- שנה את:
  - **סף זיהוי** - כמה פעמים עד התראה (1-10)
  - **חלון זמן** - בתוך כמה שניות (10-300)
  - **התראות** - הפעל/כבה overlay, צלילים, רטט
  - **מילות מפתח** - הוסף/הסר מילים

### 4. תובנות
- עבור לטאב **"תובנות"**
- ראה:
  - כמה פעמים קרא לך הילד היום
  - כמה זמן היית במסך
  - מה אחוז התגובה שלך
  - סטטיסטיקות שבועיות

## 🔧 פתרון בעיות

### האפליקציה לא נפתחת
1. וודא שיש לך Windows 10 version 19041+
2. בדוק אם יש קובץ `HereForYou.exe` בתיקייה
3. נסה להריץ כמנהל (לחץ ימין → "הרץ כמנהל")

### אין התראות
1. בדוק שהניטור פועל (כפתור ירוק)
2. עבור להגדרות ובדוק ש"התראות אקטיביות" מופעל
3. נסה להוריד את "סף זיהוי" ל-1

### מסד הנתונים לא נשמר
1. בדוק שיש הרשאות כתיבה ל-`%LOCALAPPDATA%`
2. אם הרצת כמנהל, מסד הנתונים יהיה במיקום אחר

### הרטט לא עובד
- רטט עובד רק במכשירים עם תמיכה (טאבלטים/לפטופים מסוימים)
- ב-Desktop רגיל לא יהיה רטט

## 🎨 התאמה אישית

### שינוי שפת ממשק
בגרסה נוכחית: עברית בלבד
בגרסה הבאה: אנגלית + עברית עם בחירה

### הוספת מילות מפתח
1. עבור ל"הגדרות"
2. גלול ל"מילות מפתח"
3. הקלד מילה חדשה ולחץ "הוסף"
4. למשל: "ילד", "בוא", "עזור"

### שינוי שם משתמש
1. עבור ל"הגדרות"
2. מצא "שם משתמש"
3. הקלד את השם שלך
4. ישמר אוטומטית

## 📦 גיבוי ושחזור נתונים

### גיבוי ידני
1. סגור את האפליקציה
2. העתק את הקובץ:
   ```
   %LOCALAPPDATA%\Packages\[package-name]\LocalState\hereforyou.db3
   ```
3. שמור במיקום בטוח (OneDrive, USB, וכו')

### שחזור
1. סגור את האפליקציה
2. החלף את `hereforyou.db3` עם הגיבוי שלך
3. פתח את האפליקציה - כל הנתונים יחזרו!
