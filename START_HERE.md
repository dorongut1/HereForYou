## 🪟 אפשרות 1: הרץ על Windows (הכי קל!)

### מה צריך:
- ✅ Windows 10/11
- ✅ זהו. זה הכל!

### איך:
1. **פתח PowerShell**
2. **נווט לתיקייה:**
   ```powershell
   cd C:\Disk1\HereForYou
   ```
3. **הרץ:**
   ```powershell
   dotnet run -f net10.0-windows10.0.19041.0
   ```

**או פשוט לחץ כפול על:**
```
bin\Release\net10.0-windows10.0.19041.0\win-x64\publish\HereForYou.exe
```

✅ **זהו! האפליקציה תיפתח.**

## 📱 אפשרות 2: הרץ על Android (דרך אמולטור)

### למה אמולטור?
- אין לך צורך בטלפון אנדרואיד
- רואה את האפליקציה בדיוק כמו בטלפון
- טוב לבדיקות ופיתוח

### מה צריך:
1. **Android Studio** (חינמי)
2. **Android Emulator** (בא עם Android Studio)

### איך:

#### 📖 **קרא את המדריך המפורט:**
➡️ **[INSTALL_EMULATOR_STEP_BY_STEP.md](INSTALL_EMULATOR_STEP_BY_STEP.md)**

הוא מסביר בדיוק:
- ✅ איך להוריד Android Studio
- ✅ איך להתקין אותו
- ✅ איך ליצור emulator
- ✅ איך להריץ את HereForYou עליו

זמן משוער: **30-40 דקות** (כולל הורדות)
